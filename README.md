# Moview Review App

## Descripción

Aplicación móvil de películas. Permite a los usuarios registrarse, iniciar sesión y gestionar favoritos.

## Estructura del proyecto

```
backend/                # Backend de la aplicación (Flask)
flutter_movie_app/      # Frontend de la aplicación (Flutter)
MacMovieApp/            # Frontend de la aplicación (Swift)
```
