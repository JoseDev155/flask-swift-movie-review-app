# Flutter Movie App

Aplicación de películas en Flutter.

## Overview

La apliación es **SPA**, podremos:

* Buscar películas
* Tener categorías de películas, siendo las 2 siguientes:
  * *Popular*
  * *Upcoming*
* Tener más resultados conforme *scrolleamos* más
* Actuañizar la UI, haciendo clic en la iamgen de la película y actualizar el background de la aplicación con la misma

Para esto, usaremos la API de la web [TMDB](https://www.themoviedb.org/).

## Requisitos

* Tener instalado FLutter y Dart en la máquina.
* Se puede usar la versión web y usar **Edge** en lugar de un emulador móvil.

## Instalar dependencias

* Nos posicionamos en la raíz del proyecto, donde se encuentra el `pubspec.yaml`
* Ejecutamos el comando `flutter pub get` para instalar las dependencias.

## Correr la aplicación

* Nos posicionamos en el `main.dart`
* Le damos en **Run > Start Debugging > Dart & Flutter**.
