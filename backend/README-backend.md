# Movie Review API - Backend

## Tecnologías Utilizadas

* Python `3.14.4`
* Flask
* SQLAlchemy
* Anaconda (Python Distribution)

## Instalar dependencias

Con Anaconda:

```bash
conda create -n flask-api python=3.14.4
conda activate flask-api
pip install -r requirements.txt
```

Con Python (asegurarse que sea la misma versión de arriba):

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Ejecutar la aplicación

```bash
cd backend
python main.py
```
